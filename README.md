# Hatırlatıcı Uygulaması (Reminder App)

Bu proje, Flask tabanlı bir hatırlatıcı API uygulamasıdır. Docker ile konteynerize edilmiştir.

## Docker İmajı Oluşturma (Build)
Proje dizininde aşağıdaki komutu çalıştırarak imajı oluşturabilirsiniz:

`docker build -t reminder-app .`

## Çalıştırma
Docker Compose kullanarak projeyi ayağa kaldırmak için:

`docker-compose up -d`
